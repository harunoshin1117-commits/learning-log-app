# 勉強ログアプリ

毎日学んだことを記録できる、HTML / CSS / JavaScript だけのシンプルなWebアプリです。

## 機能

- 学習内容の入力
- カテゴリ複数選択
  - JavaScript
  - Codex
  - Git
  - AI活用
  - その他
- 「その他」カテゴリの補足入力
- 今日の学習ログ保存
- 学習時間の入力
- 保存済みログ一覧表示
- 保存済みログの編集
- 更新日時の表示
- カテゴリ絞り込み
- キーワード検索
- ログ削除
- 今日学んだ数の表示
- CSVエクスポート
- 3ページ構成
  - 記録ページ
  - カレンダーページ
  - ログ一覧ページ
- カレンダーで日別学習時間を色分け表示
- 今月の学習時間、一番学んだ日の表示
- localStorageへの保存
- スマホ対応レイアウト

## 使い方

1. VS Codeでこのフォルダを開きます。
2. `index.html` をGo Liveで開きます。
3. カテゴリを1つ以上選び、学習時間と学習内容を入力して「保存する」を押します。
4. 「その他」を選んだ場合は、その他の内容も入力できます。
5. 「カレンダー」ページで月ごとの学習時間や一番学んだ日を確認できます。
6. 「ログ一覧」ページで保存済みログを表示できます。
7. カテゴリ、その他の内容、学習内容をキーワードで絞り込めます。
8. 保存済みログは「編集」ボタンで内容を更新できます。
9. 不要なログは「削除」ボタンで削除できます。
10. 「CSV」ボタンで保存済みログをCSVファイルとして出力できます。

保存データはブラウザのlocalStorageに保存されます。別のブラウザや別の端末には自動同期されません。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルをリポジトリに追加してpushします。
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. `Pages` を開きます。
5. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
6. `Branch` で公開したいブランチを選び、フォルダは `/root` を選びます。
7. `Save` を押します。
8. 表示されたGitHub PagesのURLにアクセスします。

このアプリは相対パスで作っているため、GitHub Pagesのサブパス公開でも動きやすい構成です。
